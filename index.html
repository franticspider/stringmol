Welcome to the Stringmol github pages

Stringmol is an automata chemistry, developed at the University of York from 2009. It is the subject of several academic papers in the Artificial Life research domain. These github pages contain the source code and config files for Stringmol, which can be compiled and run on most Linux OSs.
